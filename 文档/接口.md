## 收藏书签相关

### 大类

* 添加 **post** `v1/bookmark/type/add`

参数名     |  类型    | 是否必要  | 说明              
------     | ------  | ------   | ------          
name       | String  | 是       | 书签分类名         
icon       | String  | 是       | 书签类图标         
show       | Bool    | 是       | 是否显示        
createTime | date    | 是       | 创建时间
changeTime | date    | 是       | 修改时间

* 删除 **post** `v1/bookmark/type/delete`

> 常用分类不能删除

参数名 |  类型    | 是否必要  | 说明              
------ | ------  | ------   | ------         
id     | int     | 是       | 删除大类的id       

* 修改 **post** `v1/bookmark/type/change`

修改书签

参数名     |  类型    | 是否必要  | 说明              
------     | ------  | ------   | ------       
id         | int     | 是       | 大类的id           
name       | String  | 是       | 书签分类名         
icon       | String  | 是       | 书签类图标         
show       | Bool    | 是       | 是否显示     
changeTime | date    | 是       | 修改时间   

修改显示方式

参数名     |  类型    | 是否必要  | 说明              
------     | ------  | ------   | ------       
id         | int     | 是       | 大类的id           
name       | String  | 是       | 书签分类名        

* 查询 **post** `v1/bookmark/type/query`

参数名          |  类型   | 是否必要  | 说明              
------         | ------  | ------   | ------          
sideDisplay    | int     | 是       | 控制侧边书签显示方式，0：隐藏，1：仅显示图标，2：仅显示入口   
commonDisplay  | int     | 是       | 控制常用书签是否显示，0：正常显示，1：隐藏，但鼠标覆盖显示，2：完全隐藏   
position       | int     | 是       | 侧边书签位置，0,1,2,3 上右下左   
 

### 单个书签

* 添加 **post** `v1/bookmark/item/add`

参数名      |  类型   | 是否必要 | 说明              
------     | ------  | ------  |------         
parentid   | int     | 是      | 上级书签分类id       
name       | String  | 是      | 书签名              
icon       | String/image  | 是      | 书签图标，字符串（从前端svg库选择）或者图片（单独上传）
style      | String  | 是      | 书签背景颜色等css样式 
url        | String  | 是      | 书签的链接 
createTime | date    | 是      | 创建时间
changeTime | date    | 是      | 修改时间

* 删除 **post** `v1/bookmark/item/delete`

参数名      |  类型   | 是否必要 | 说明              
------     | ------  | ------  |------         
id         | int     | 是      | 书签id

* 修改 **post** `v1/bookmark/item/change`

参数名      |  类型   | 是否必要 | 说明              
------     | ------  | ------  |------         
parentid   | int     | 是      | 上级书签分类id       
id         | int     | 是      | 书签id       
name       | String  | 否      | 书签名              
icon       | String  | 否      | 书签图标   
style      | String  | 否      | 书签背景颜色等css样式 
url        | String  | 否      | 书签的链接 
changeTime | date    | 是      | 修改时间

* 查询 **post** `v1/bookmark/item/query`

参数名      |  类型   | 是否必要 | 说明              
------     | ------  | ------  |------         
parentid   | int     | 是      | 上级书签分类id       
id         | int     | 是      | 书签id

## 搜索引擎相关

### 大类

* 添加 **post** `v1/engine/type/add`

参数名      |  类型   | 是否必要 | 说明              
------     | ------  | ------  |------         
name       | String  | 是      | 搜索类型名             
active     | Bool    | 是      | 是否激活可用 
current    | Bool    | 是      | 当前搜索所属项，点击搜索该分类仅该分类为true，待讨论

------     | ------  | ------  |------      
id         | int     | 是      | 修改项id        
name       | String  | 是      | 新名字             
active     | Bool    | 是      | 是否激活可用
