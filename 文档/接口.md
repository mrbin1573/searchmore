> 创建时间:2019年1月18日
> 所有请求使用**post**，数据格式**json**
## 账号相关

### 登录

微信扫一扫登录

### 登出 
**post** `/v1/logout`

## 收藏书签相关

### 大类

* 添加 **post** `v1/bookmark/type/add`

参数名     |  类型    | 是否必要  | 说明              
------     | ------  | ------   | ------          
name       | String  | 是       | 书签分类名         
icon       | String  | 是       | 书签类图标         
order      | int     | 是       | 显示顺序，默认在最后
show       | Bool    | 是       | 是否显示        
createTime | date    | 是       | 创建时间
changeTime | date    | 是       | 修改时间

* 删除 **post** `v1/bookmark/type/delete`

> 常用分类不能删除

参数名 |  类型    | 是否必要  | 说明              
------ | ------  | ------   | ------         
id     | int     | 是       | 删除大类的id       

* 修改 **post** `v1/bookmark/type/change`

参数名     |  类型    | 是否必要  | 说明              
------     | ------  | ------   | ------       
id         | int     | 是       | 大类的id           
name       | String  | 是       | 书签分类名         
icon       | String  | 是       | 书签类图标         
order      | int     | 是       | 显示顺序，默认在最后
show       | Bool    | 是       | 是否显示     
changeTime | date    | 是       | 修改时间      

* 查询 **post** `v1/bookmark/type/query`

参数名 |  类型    | 是否必要  | 说明              
------ | ------  | ------   | ------          
id     | int     | 是       | 大类的id   
 

### 单个书签

* 添加 **post** `v1/bookmark/item/add`

参数名      |  类型   | 是否必要 | 说明              
------     | ------  | ------  |------         
parentid   | int     | 是      | 上级书签分类id       
name       | String  | 是      | 书签名              
icon       | String/image  | 是      | 书签图标，字符串（从前端svg库选择）或者图片（单独上传）
order      | int     | 是      | 显示顺序，默认在最后 
style      | String  | 是      | 书签背景颜色等css样式 
url        | String  | 是      | 书签的链接 
createTime | date    | 是      | 创建时间
changeTime | date    | 是      | 修改时间

* 删除 **post** `v1/bookmark/item/delete`

参数名      |  类型   | 是否必要 | 说明              
------     | ------  | ------  |------         
id         | int     | 是      | 书签id

* 修改 **post** `v1/bookmark/item/change`

参数名      |  类型   | 是否必要 | 说明              
------     | ------  | ------  |------         
parentid   | int     | 是      | 上级书签分类id       
id         | int     | 是      | 书签id       
name       | String  | 否      | 书签名              
icon       | String  | 否      | 书签图标            
order      | int     | 否      | 显示顺序
style      | String  | 否      | 书签背景颜色等css样式 
url        | String  | 否      | 书签的链接 
changeTime | date    | 是      | 修改时间

* 查询 **post** `v1/bookmark/item/query`

参数名      |  类型   | 是否必要 | 说明              
------     | ------  | ------  |------         
parentid   | int     | 是      | 上级书签分类id       
id         | int     | 是      | 书签id

## 搜索引擎相关

### 大类

* 添加 **post** `v1/engine/type/add`

参数名      |  类型   | 是否必要 | 说明              
------     | ------  | ------  |------         
name       | String  | 是      | 搜索类型名             
active     | Bool    | 是      | 是否激活可用 
current    | Bool    | 是      | 当前搜索所属项，点击搜索该分类仅该分类为true，待讨论
order      | int     | 是      | 排列顺序
createTime | date    | 是      | 创建时间
changeTime | date    | 是      | 修改时间

* 删除 **post** `v1/engine/type/delete`

参数名      |  类型   | 是否必要 | 说明              
------     | ------  | ------  |------         
id         | int     | 是      | 删除项id         

* 修改 **post** `v1/engine/type/change`

参数名      |  类型   | 是否必要 | 说明              
------     | ------  | ------  |------      
id         | int     | 是      | 修改项id        
name       | String  | 是      | 新名字             
active     | Bool    | 是      | 是否激活可用 
order      | int     | 是      | 排列顺序
changeTime | date    | 是      | 修改时间

* 查询 **post** `v1/engine/type/query`

参数名      |  类型   | 是否必要 | 说明              
------     | ------  | ------  |------      
id         | int     | 是      | 项id        

### 单个搜索引擎

* 添加 **post** `v1/engine/item/add`

参数名      |  类型   | 是否必要 | 说明              
------     | ------  | ------  |------         
parentid   | int     | 是      | 上级引擎分类id       
name       | String  | 是      | 引擎名字              
browserUrl | String  | 是      | 引擎搜索链接 
resultSrc  | String  | 是      | 引擎搜索结果链接，待讨论 
order      | int     | 是      | 显示顺序，默认在最后 
icon       | String  | 否      | 非必须，引擎图标，必须从前端库上传，为了搜索结果页面统一
active     | Bool    | 是      | 是否激活可用状态 
current    | Bool    | 是      | 是否为当前选中显示搜索结果页面，待讨论
resultIframeStyle| String| 是  | 去除该结果页面的样式不统一
createTime | date    | 是      | 创建时间
changeTime | date    | 是      | 修改时间

* 删除 **post** `v1/engine/item/delete`

参数名      |  类型   | 是否必要 | 说明              
------     | ------  | ------  |------         
id         | int     | 是      | 引擎id    

* 修改 **post** `v1/engine/item/change`

参数名      |  类型   | 是否必要 | 说明              
------     | ------  | ------  |------  
parentid   | int     | 是      | 上级引擎分类id         
id         | int     | 是      | 修改项id        
name       | String  | 是      | 引擎名字              
browserUrl | String  | 是      | 引擎搜索链接 
resultSrc  | String  | 是      | 引擎搜索结果链接，待讨论 
order      | int     | 是      | 显示顺序，默认在最后 
icon       | String  | 否      | 非必须，引擎图标，必须从前端库上传，为了搜索结果页面统一
active     | Bool    | 是      | 是否激活可用状态 
current    | Bool    | 是      | 是否为当前选中显示搜索结果页面，待讨论
resultIframeStyle| String| 是  | 去除该结果页面的样式不统一
changeTime | date    | 是      | 修改时间


* 查询 **post** `v1/engine/item/query`

参数名      |  类型   | 是否必要 | 说明              
------     | ------  | ------  |------      
id         | int     | 是      | 项id
